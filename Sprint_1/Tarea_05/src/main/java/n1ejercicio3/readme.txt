Para ejecutar por consola:

1- abrir terminal en direccion un nivel por encima del package, (/main/java) en este caso.
2- escribir -> javac n1ejercicio3/DirectorySorter3.java (Esto lo hace ejecutable)
3- escribir -> java n1ejercicio3.DirectorySorter3 (lo ejecuta)
4- escribir ruta del directorio
5- el programa creará un archivo .txt en la dirección actual para ver el resultado
