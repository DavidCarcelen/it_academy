Para ejecutar por consola:

1- abrir terminal en direccion un nivel por encima del package, (/main/java) en este caso.
2- escribir -> javac n1ejercicio4/DirectorySorter4.java (Esto lo hace ejecutable)
3- escribir -> java n1ejercicio4.DirectorySorter4  (lo ejecuta)
4- elejir directorio (1) o archivo txt (2)
5- escribir ruta del directorio o del archivo.
6- el directorio lo elcribirá en un archivo txt, el archivo mostrará el contenido por la terminal.
